## 2. 業界研究 — ビルメンテナンス・設備管理

### 2-1. 業界全体のトレンド

| トレンド | 内容 |
|---|---|
| 慢性的人手不足 | 団塊世代引退 + 若手不足 = **40-50 代前半まで採用ニーズ大** |
| ホワイト化 | 大手ビルメン（東京美装・日本管財・大星ビル等）で待遇改善が進む |
| **資格保有者の希少化** | ビル管 / 電工 / 電験保有者は転職市場で**売り手市場** |
| データセンター需要 | 半導体・AI 投資で DC 増加 = 高度設備管理の需要拡大 |
| 自動化進行 | BAS（ビル自動化システム）の高度化、保全業務もデータ駆動化 |
| 商業施設の再編 | EC シフトで規模縮小もあるが、**観光地型アウトレットは堅調**（木更津該当）|

### 2-2. 設備管理（巡回点検）の仕事の本質

> 業界用語で言えば「**ビルの神経系・血管系の整備**」（AGC 計装と同じ表現が成立）

- **電気設備**（受変電設備・配電盤・自家発電・照明）
- **空調設備**（セントラル空調・パッケージ・換気・ファンコイル）
- **給排水・衛生設備**（給水ポンプ・排水・受水槽・浄化槽）
- **防災設備**（消火栓・スプリンクラー・自動火災報知・防排煙）
- **冷凍冷蔵設備**（フードコート・店舗向け冷凍機）
- **昇降機**（エレベーター・エスカレーター — 専門業者連携）

→ AGC の計装（プラント機器の維持）と異なり、**多種多様な設備を「広く浅く」管理**するのが特徴。「狭く深い」AGC vs 「広く浅い」東京美装。

### 2-3. 同業他社との比較（千葉県内勤務想定）

| 社名 | 規模 | 千葉県内拠点 | 備考 |
|---|---|---|---|
| 日本管財 | 売上 1,238 億 | あり | 東証プライム上場・業界最大手の一角 |
| イオンディライト | 売上 3,254 億 | あり（イオン関連施設）| 東証プライム・イオングループ |
| **東京美装興業** | **売上 575 億** | **木更津金田東等** | **非上場・独立系・本求人** |
| 大星ビル管理 | 売上 470 億 | あり | NTT 系 |
| 第一ビルメンテナンス | 売上 200 億 | 限定 | 中堅 |
| ALSOK ファシリティーズ | ALSOK 子会社 | あり | 警備系統合 |

→ **東京美装は業界中堅から上位**。日本管財・イオンディライトより小規模だが、独立系で**経営判断が早い・配属希望反映度が比較的高い**との評判。

### 2-4. ビル管理 4 点セット（資格戦略の中心）

§11-5 戦略の中核資格群。東京美装で実務経験 + 資格取得を並行できる。

| 資格 | 用途 | 東京美装での手当 |
|---|---|---|
| **第 2 種電気工事士** | 電気設備の保守・小規模工事 | **月 5,000 円** |
| **危険物取扱者乙種 4 類** | ボイラー燃料・自家発電の燃料管理 | 別途（消防設備士枠で）|
| **冷凍機械責任者 3 種** | 冷凍空調設備の責任者 | **月 3,000 円** |
| **2 級ボイラー技士** | ボイラー設備の保守（大型施設は必要）| 別途 |
| **建築物環境衛生管理技術者（ビル管）**| 3,000m² 超の特定建築物の管理責任者 | **月 7,000 円** |
| **第 3 種電気主任技術者（電験 3 種）**| 高圧受電設備の責任者・需要極めて高い | **月 15,000 円** |
| **第 1・2 種電気主任技術者（電験 1・2 種）**| 大規模高圧受電・特高受電の責任者 | **月 30,000 円** |

→ 累計上限月 70,000 円 = **年 84 万円**。30 代後半までに 4 点セット完取で月 15-25,000 円、40 代でビル管・電験 3 種まで取れば月 30-50,000 円が現実的。

   3-1. 企業への関心（リーディングカンパニー / 3 事業シナジー / 木更津金田東）
   3-2. 業務への関心（設備管理の本質, 大型商業施設運営の社会的意義）
   3-3. 地域への定着意思（結婚転居で袖ヶ浦定住、木更津 20-30 分）
[4] 自身の強み + 入社後の貢献
   4-1. 学習意欲（IT パスポート等の取得実績）
   4-2. 正確性・調整力（財務省で培った）
   4-3. 資格取得計画（第二種電気工事士・建築物環境衛生管理技術者等）
[5] 長期貢献ビジョン（雇用延長 65 歳まで）

## 5. 言ってはいけないこと

| ❌ NG | 理由 |
|---|---|
| YouTube・副業の話 | 絶対原則（LIFE_PLAN.md §15-8）。ライフプラン全体に影響 |
| 給与・資格手当が魅力 | 短期志向と見られる、特に資格手当 7 万に飛びついた印象は要警戒 |
| 「夜勤少なそう」「楽そう」 | 業務軽視と見られる、シフト [3] 通し勤の存在を承知の上での応募と見せる |
| 「未経験ですが頑張ります」のみ | 抽象的すぎる、東京美装らしさがない |
| 「お客様のために」のみ | カスタマー職と勘違いされる、設備管理はバックヤード |
| 警備業務への言及 | 設備管理応募で警備に触れると配属誤解の元 |

## 6. 言うと評価が上がること

> **注意**: 「リーディングカンパニー」「3 事業のシナジー」等の固有表現は
> 履歴書では使わない（過剰準備感が出る）。**面接で会話の流れで自然に出せたら**
> プラスに働く程度。基本は「ニュース・求人票で読んだ範囲の言葉」で語る。

| ✅ 言うと良い | 効果 |
|---|---|
| ビル管理リーディングカンパニー | 業界研究の証拠、自然な範囲 |
| 「清掃・設備・警備の総合管理」 | 3 事業シナジーへの理解 |
| 「1957 年からの長い歴史」 | 会社案内レベルの知識、自然 |
| 「木更津金田東で大型商業施設」 | 配属先への関心 |
| 「1 施設チーム制」「OJT」への理解 | 求人票を読んだ証拠、業務理解 |
| **第二種電気工事士・建築物環境衛生管理技術者**の取得意欲 | 資格取得支援制度を理解、即戦力化への意欲 |
| 「設備の地道な維持が施設運営の前提」 | 抽象的だが東京美装の業務本質と整合 |
| **家族と袖ヶ浦で長く暮らしながら、手に職を磨きたい** | **年配 HR に強く効く、誠実さの証明**（前職を否定せず家庭志向と職業志向を統合）|
