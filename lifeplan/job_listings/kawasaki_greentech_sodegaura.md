### 致命的: 完全夜勤の存在

| 観点 | 内容 |

| 生体リズム | 早番 → 中番 → 夜勤の 3 交代でメラトニン分泌乱れ |

## 結論・推奨アクション

### スコア判定

**応募見送り推奨**（順位 8、加重スコア 2.20、床下回り）。

### 見送りの根拠

1. **加重スコア 2.20 で床（3.10）を下回る**（重み付け再評価でも同様）
2. **完全夜勤 22:30-7:30 が致命的**、§11-5 で棄却した夜勤類型に該当
3. **経済性も中立で床 -370 万**、楽観でも +290 万に留まる
4. **子育て期で妻退職リスク増**（§16 妻フル/時短シナリオ破壊）
5. ALSOK と同類のシフト致命傷を抱える

### 業界視点での価値

ユーザー所感「こういうタイプなら意外と選択肢は多い」への分析:

| 観点 | 内容 |
|---|---|
| **袖ヶ浦・木更津周辺の同業他社** | 東京電力 / JERA / 出光 / 日本製鉄 / 各種化学プラント等多数 |
| 共通課題 | **多くが 3 交代 / 夜勤含む**、子育て期との両立は業界共通課題 |
| 例外（夜勤なしの選択肢）| AGC 本体（フレックス）/ 東京美装（昼勤希望可）等の限定候補 |
| 結論 | **「袖ヶ浦近郊 + 設備管理 + 大手」**ジャンルは選択肢多いが、**「夜勤なし」**で絞ると候補は激減 |

### §11-5 戦略との関係

本求人は §11-5 「ビル管理 4 点セット + 設備就業」のうち**設備就業の具体例**として強い:
- 電工 1・2 種、電気主任技術者、エネ管、ボイラー・タービン等の取得支援
- 川重子会社で大手安定基盤
- ただし**取得後に転職前提**なら有用、現職継続前提なら夜勤負担が重い

→ **「資格取得して数年で転職」戦略**ならアリ、**「定年まで在籍」戦略**なら健康面で困難。

### 加齢時の代替候補としての価値

50-60 代以降のセカンドキャリア候補としては:
- 通勤超近距離 + 大手安定 + 再雇用制度
- ただし 60 代で 3 交代夜勤を続ける体力は要慎重判断

### 撤退基準（仮に応募する場合）

- 試用 3 ヶ月で夜勤頻度が月 4 回超 → 撤退
- 健康指標悪化（睡眠の質・血圧）→ 撤退
- 妻が「夜勤頻度を減らしてほしい」訴え → 撤退検討
