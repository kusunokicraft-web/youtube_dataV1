### 1-3. 中期経営計画「AGC plus-2026」（最重要）

> 求人票では触れられないが、面接で言及すると「業界研究してる」と評価される。

#### 基本戦略
- **「両利きの経営」**を推進（既存事業の深化 + 新規事業の探索）
- **パフォーマンスケミカルズ（フッ素系材料）を戦略事業**として位置付け
- 市況変動に強く、資産効率・成長性・炭素効率の高い事業ポートフォリオ構築

#### 財務目標（2026 年）
- 営業利益 **2,300 億円**達成
- **ROE 8% 以上**を安定的に達成
- 生産能力拡大投資は 2025 年で一巡 → **以降は投資効果発現フェーズ**

## 2. 業界研究 — 化学プラント設備保全（計装）

### 2-1. 業界全体のトレンド

| トレンド | 内容 |
|---|---|
| 慢性的人手不足 | 団塊世代引退 + 製造業敬遠 = 中堅技術者の空白 |
| ホワイト化 | 採用ブランド戦略で待遇向上 |
| 自動化進行 | DCS / PLC の高度化、保全業務もデータ駆動化 |
| 半導体特需 | 半導体関連向け化学品需要増 = 千葉工場の主戦場 |
| 脱炭素 | カーボンニュートラル対応で工場改修需要 |

### 2-2. 計装エンジニアの仕事の本質

> 業界用語で言えば「**プラントの神経系・血管系の整備**」

- DCS（分散制御システム）/ PLC（プログラマブルロジックコントローラ）の現場機器
- 計測機器（流量・温度・圧力・レベル・pH 等）
- 自動弁（コントロールバルブ）
- 電気設備（モーター制御盤・配電・防爆機器）

| なぜ設備保全に興味を？ | **「財務省時代に公務員宿舎の管理で電気・ガス・水道の整備発注に関わり、現場で設備を支える仕事に関心を持った」**（実体験） |
| 公務員宿舎の管理ってどんな仕事？ | 自衛隊員などが利用する宿舎の電気・ガス・水道の整備発注、修繕の手配等を担当 |
| 金融教育アプリの開発内容は？ | 監修と金融教育コンテンツ作成、業務効率化のためにマクロも担当 |
| なぜ法学部から技術職？ | 「設備への興味は財務省時代から持っていた、自分で学んで身につける姿勢は IT 資格取得実績が証拠」 |
| 千葉に長く住む？ | 「袖ヶ浦に住居を構え、長期的にこの地域で働きたい」と簡潔に |
| 体力面は大丈夫？ | 「健康診断は問題なし、現場作業への適性は十分」と言い切る |
| 家族構成は？お子さんは？ | 「結婚したばかりで、これから家族を築いていきたいと思っています」と素直に。**「子どもにも誇れる仕事を」志望動機と整合させる** |
| 5 年 / 10 年後はどうしたい？ | 「現場のスペシャリストとして基礎技術を確立し、家族と支え合いながら長くこの地域で働いていきたい」 |

## 5. 言ってはいけないこと

| ❌ NG | 理由 |
|---|---|
| YouTube・副業の話 | 絶対原則（LIFE_PLAN.md §15-8）。ライフプラン全体に影響 |
| 給与・待遇が魅力 | 短期志向と見られる |
| ホワイト企業ランクが魅力 | 楽な仕事を求めていると見られる |
| 「未経験ですが頑張ります」のみ | 抽象的すぎる、AGC らしさがない |
| 「お客様のために」のみ | カスタマー職と勘違いされる、設備保全はバックヤード |

## 6. 言うと評価が上がること

> **注意**: 戦略名「AGC plus-2026」「パフォーマンスケミカルズ」等の固有名詞は
> 履歴書では使わない（過剰準備感が出る）。**面接で会話の流れで自然に出せたら**
> プラスに働く程度。基本は「ニュースで知った範囲の言葉」で語る。

| ✅ 言うと良い | 効果 |
|---|---|
| フッ素関連の素材 / 半導体・自動車向け | ニュース範囲の業界理解 |
| 「千葉工場で新しい設備が動き始めている」 | 配属先への関心 |
| 100 年以上の歴史・世界トップクラス | 会社案内レベルの知識、自然 |
| 「2 人 1 組」「OJT」への理解 | 求人票を読んだ証拠、業務理解 |
| 第二種電気工事士の取得意欲 | 資格取得支援制度を理解、即戦力化への意欲 |
| 社会基盤を支える素材 | 抽象的だが AGC の自負と整合 |
| **家庭を持って腰を据えたい / 子どもに誇れる仕事** | **年配 HR に強く効く、誠実さの証明**（うっすら一文でにじませる程度） |

### 6-1. 面接で深掘りされたら使える知識（履歴書には書かない）

| 言い方 | 元ネタ |
|---|---|
| 「中期経営計画でフッ素事業に注力されていると拝見しました」 | AGC plus-2026 |
| 「次世代冷媒の HFO-1234yf の製造設備が稼働開始したとニュースで」 | 350 億円投資のニュース |
| 「両利きの経営という考え方を取られていると」 | IR 資料 |

→ 履歴書では「フッ素事業に注力」「新しい設備が動き始めている」レベル、
面接で深掘られたら IR 用語を 1-2 個出せる、というレイヤー構造が自然。

## 7. 面接でよく聞かれる質問と回答方針

| 質問 | 回答方針 |
|---|---|
| なぜ未経験でこの業界に？ | 「手に職」「長期キャリア」「業界の安定性」を語る |
| なぜ前職を辞めたい / 辞めた？ | ネガティブを避け、「自分の手で何かを動かす仕事をしたい」とポジティブ転換 |
| AGC の何を知っているか？ | AGC plus-2026、千葉工場の役割、フッ素事業の戦略性 |
| 設備保全の何が魅力？ | 「プラントの神経を支える地道で重要な仕事」「2 人 1 組の安全文化」 |
| 残業・夜間呼び出しは大丈夫？ | 「業務上必要であれば対応します」（具体的な不安は隠す）|
| 5 年後のキャリアイメージ | 「現場のスペシャリストとして基礎技術を確立し、計装スキルを深めたい」|
| 質問はありますか？（逆質問）| 後述 |

### 7-1. 逆質問の例（評価が上がる質問）

| 質問例 | 効果 |
|---|---|
| 千葉工場の中で計装チームが直面している技術的課題は？ | 業務理解の深さアピール |
| 入社 1-3 年目の方は具体的にどのようなステップで成長していますか？ | 長期キャリア志向アピール |
| AGC plus-2026 で千葉工場に求められている役割はどう変化していますか？ | 経営戦略への関心 |
| 計装担当として持っていると評価される資格はありますか？ | 自己投資意欲 |
| OB の方で印象的なキャリアを歩まれた方はいますか？ | カルチャーフィット |

### 7-2. 逆質問で避けるべきもの

| ❌ 避ける | 理由 |
|---|---|
| 残業時間・休日について | 求人票で確認済の体で進める |
| 給与の詳細 | 評価面談などで聞く話 |
| 副業・兼業の可否 | 警戒される |
| すぐ家に帰れますか系 | 仕事への意欲を疑われる |
