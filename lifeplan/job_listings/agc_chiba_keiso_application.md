## 1. 企業研究 — AGC株式会社

### 1-2. 事業ポートフォリオ（4 セグメント）

| 事業 | 内容 | 強み |
|---|---|---|
| **ガラス** | 建築用・自動車用ガラス | 世界トップクラスシェア |

→ **多角化で景気変動に強い**ビジネスモデル。100 年磨いたガラス製造ノウハウを派生させて多領域展開。

### 1-3. 中期経営計画「AGC plus-2026」（最重要）

> 求人票では触れられないが、面接で言及すると「業界研究してる」と評価される。

#### 基本戦略
- **「両利きの経営」**を推進（既存事業の深化 + 新規事業の探索）
- **パフォーマンスケミカルズ（フッ素系材料）を戦略事業**として位置付け
- 市況変動に強く、資産効率・成長性・炭素効率の高い事業ポートフォリオ構築

#### 財務目標（2026 年）
- 営業利益 **2,300 億円**達成
- **ROE 8% 以上**を安定的に達成
- 生産能力拡大投資は 2025 年で一巡 → **以降は投資効果発現フェーズ**

### 1-4. 千葉工場の戦略的位置付け（要・面接で言及）

|---|---|

| 主力製品 | **フッ素製品**（半導体・航空機・自動車向け）|
| 大型投資 | **2023 年に 350 億円投資**を発表 |

## 2. 業界研究 — 化学プラント設備保全（計装）

### 2-1. 業界全体のトレンド

| トレンド | 内容 |
|---|---|
| 慢性的人手不足 | 団塊世代引退 + 製造業敬遠 = 中堅技術者の空白 |
| ホワイト化 | 採用ブランド戦略で待遇向上 |
| 自動化進行 | DCS / PLC の高度化、保全業務もデータ駆動化 |
| 半導体特需 | 半導体関連向け化学品需要増 = 千葉工場の主戦場 |
| 脱炭素 | カーボンニュートラル対応で工場改修需要 |

### 2-2. 計装エンジニアの仕事の本質

> 業界用語で言えば「**プラントの神経系・血管系の整備**」

- DCS（分散制御システム）/ PLC（プログラマブルロジックコントローラ）の現場機器
- 計測機器（流量・温度・圧力・レベル・pH 等）
- 自動弁（コントロールバルブ）
- 電気設備（モーター制御盤・配電・防爆機器）

## 3. 志望動機案

> 個人プロファイル参照: `lifeplan/profile.md`
> （氏名: 石田陸人, 26 歳, 財務省 → グリーンモンスター → 転職活動中, 結婚で袖ヶ浦転居）

### 3-1. 履歴書用 Pattern A（経歴の流れ重視型・約 310 字）

```
新卒で財務省に入り、財務専門官として2年9ヶ月、税務や利害調整の仕事に
携わりました。その後民間に移ってマクロを使った業務効率化に1年ほど
取り組みましたが、結婚を機に袖ヶ浦へ転居して一区切りつきました。
文書や調整中心の仕事を続けるうち、もっと手を動かして目に見えるものを
支える仕事をしたいという気持ちが強くなり、今回の転職を決めました。
住まいの近くで腰を据えて働ける場を探していたとき、貴社千葉工場で
半導体や自動車向けのフッ素関連素材の生産が拡大していると知り、
ぜひその現場で長く働きたいと感じています。
```

**狙い**: 経歴を素直に語り、結婚転居で千葉定住の根拠を示す
**強み**: 「財務省 → ベンチャー → 化学プラント」の方向転換に違和感のない説明
**推奨度**: ◎ **本命候補**

### 3-2. 履歴書用 Pattern B（強み訴求型・約 290 字）

```
これまで「現場で必要なことを自分で学んで、周りの役に立つ」というやり方で
仕事をしてきました。財務省では利害調整、前職ではマクロを覚えて業務効率化
に取り組み、その都度足りない知識を埋めるためにITパスポートや情報セキュリ
ティマネジメント、簿記なども取得してきました。設備保全（計装）は、知識を
一段ずつ積み上げて長く「手に職」を持てる仕事だと聞いています。100年以上
ガラスや化学品で世界の現場を支えてきた貴社で、OJTや資格取得支援を活かし
ながら第二種電気工事士などを取り、いずれは現場で頼ってもらえる人に
なりたいです。
```

**狙い**: 「自分で学んで現場を良くする」既存の自己理解を活かして AGC に紐づける
**強み**: 学習資格 4 つ（IT パスポート / 情報セキュリティマネジメント / 簿記 / 第二種電工）で**自ら学ぶ姿勢を客観的に示す**
**推奨度**: ◎ **予備候補**（A が刺さらない場合の保険）

### 3-3. 履歴書用 Pattern C（AGC 戦略結合型・約 310 字）

```
財務省での調整業務、前職でのマクロによる業務効率化と、分野は違っても
「自分で学んで現場を良くする」という働き方に手応えを感じてきました。
次は、その姿勢を活かして専門技術をしっかり身につけたいと考えています。
結婚で袖ヶ浦に住まいを構えたタイミングで貴社の募集を知り、応募させて

第二種電気工事士から少しずつ資格も増やしていきたいです。
```

**狙い**: 経歴の強み + 業界研究の深さを両立、千葉定住も自然に
**強み**: 350 億円投資の言及で**業界研究の深さを強くアピール**
**推奨度**: ○ 業界研究をしっかりした自負を示したいなら（深掘りリスク注意）
