## 1. 企業研究 — AGC株式会社

| 項目 | 内容 |
|---|---|

### 1-2. 事業ポートフォリオ（4 セグメント）

| 事業 | 内容 | 強み |
|---|---|---|
| **ガラス** | 建築用・自動車用ガラス | 世界トップクラスシェア |
| **電子** | ディスプレイ・電子部材 | 半導体関連の高機能素材 |

→ **多角化で景気変動に強い**ビジネスモデル。100 年磨いたガラス製造ノウハウを派生させて多領域展開。

### 1-3. 中期経営計画「AGC plus-2026」（最重要）

> 求人票では触れられないが、面接で言及すると「業界研究してる」と評価される。

#### 基本戦略
- **「両利きの経営」**を推進（既存事業の深化 + 新規事業の探索）
- **パフォーマンスケミカルズ（フッ素系材料）を戦略事業**として位置付け
- 市況変動に強く、資産効率・成長性・炭素効率の高い事業ポートフォリオ構築

#### 財務目標（2026 年）
- 営業利益 **2,300 億円**達成
- **ROE 8% 以上**を安定的に達成
- 生産能力拡大投資は 2025 年で一巡 → **以降は投資効果発現フェーズ**

### 1-4. 千葉工場の戦略的位置付け（要・面接で言及）

| 主力製品 | **フッ素製品**（半導体・航空機・自動車向け）|
| 大型投資 | **2023 年に 350 億円投資**を発表 |

## 2. 業界研究 — 化学プラント設備保全（計装）

### 2-1. 業界全体のトレンド

| トレンド | 内容 |
|---|---|
| 慢性的人手不足 | 団塊世代引退 + 製造業敬遠 = 中堅技術者の空白 |
| ホワイト化 | 採用ブランド戦略で待遇向上 |
| 自動化進行 | DCS / PLC の高度化、保全業務もデータ駆動化 |
| 半導体特需 | 半導体関連向け化学品需要増 = 千葉工場の主戦場 |
| 脱炭素 | カーボンニュートラル対応で工場改修需要 |

### 2-2. 計装エンジニアの仕事の本質

> 業界用語で言えば「**プラントの神経系・血管系の整備**」

- DCS（分散制御システム）/ PLC（プログラマブルロジックコントローラ）の現場機器
- 計測機器（流量・温度・圧力・レベル・pH 等）
- 自動弁（コントロールバルブ）
- 電気設備（モーター制御盤・配電・防爆機器）

## 3. 志望動機案

> 個人プロファイル参照: `lifeplan/profile.md`
> （氏名: 石田陸人, 26 歳, 財務省 → グリーンモンスター → 転職活動中, 結婚で袖ヶ浦転居）

### 3-1. 履歴書用 Pattern A（経歴の流れ重視型・約 310 字）

```
財務省で財務専門官として正確な事務と複雑な利害調整に2年9ヶ月従事した後、
前職のグリーンモンスター株式会社ではGoogle Apps Scriptを用いた業務効率化に
取り組んでまいりました。これらの経験を通じ、自分の手で何かを動かし支える
「現場」の仕事に長期的に関わりたいという思いが強まり、結婚を機に袖ヶ浦へ
転居したことを契機に、地域の基幹産業である化学プラントに目を向けました。
貴社千葉工場は中期経営計画AGC plus-2026の戦略事業を担う中核拠点であり、
その安定運転を支える設備保全（計装）として、これまで培った学習意欲と
正確性を活かして長く貢献してまいります。
```

**狙い**: 経歴の流れを自然にストーリー化、結婚転居で千葉定住の根拠を示す
**強み**: 「財務省 → ベンチャー → 化学プラント」の方向転換に違和感のない説明
**推奨度**: ◎ **本命候補**

### 3-2. 履歴書用 Pattern B（強み訴求型・約 290 字）

```
「現場で必要なことを自ら学び、周囲の役に立つ」ことが私の原動力です。
財務省での利害調整や前職でのGoogle Apps Scriptを用いた業務効率化など、
異なる分野で「現場を良くするために何が必要か」を考え、ITパスポートや
情報セキュリティマネジメント、簿記など必要な知識を自ら補ってきました。
化学プラントの設備保全（計装）は、専門知識を体系的に積み上げ、長期的に
「手に職」を磨ける仕事と理解しております。創業118年の歴史と世界トップ
クラスの技術力を持つ貴社で、丁寧なOJTと資格取得支援制度のもと、第二種
電気工事士など関連資格を着実に取得しながら、現場のスペシャリストとして
貢献してまいります。
```

**狙い**: 既存の汎用志望動機を活かしつつ AGC・設備保全に紐づける
**強み**: 学習資格 4 つ（IT パスポート / 情報セキュリティマネジメント / 簿記 / 第二種電工）を列挙して**自ら学ぶ姿勢を客観的に示す**
**推奨度**: ◎ **予備候補**（A が刺さらない場合の保険）

### 3-3. 履歴書用 Pattern C（AGC 戦略結合型・約 300 字）

```
財務省での調整業務や前職でのGASによる業務効率化など、異なる分野で
自ら学び現場を良くする働き方に充実を感じてまいりました。今後はその
姿勢を活かし、長期的に専門技術を磨ける環境で「手に職」を付けたいと

戦略事業に位置付けており、千葉工場では2023年の350億円投資による新製造
設備の稼働など、成長フェーズの中核拠点として活発に展開していらっしゃい
ます。設備保全（計装）として地域の基幹産業を現場から支え、第二種電気
工事士など関連資格の取得にも挑戦しながら長く貢献してまいります。
```

**狙い**: 経歴の強み + 業界研究の深さを両立、千葉定住も自然に
**強み**: 350 億円投資の言及で**業界研究の深さを強くアピール**
**推奨度**: ○ 業界研究をしっかりした自負を示したいなら（深掘りリスク注意）
