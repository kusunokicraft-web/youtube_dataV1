| 観点 | 内容 |

| いつ言うか | 「なぜ財務省を辞めた？」と深掘られた時、公式理由を答えた後に「もう一つ正直に申し上げますと…」と続ける |

| 整合性 | 財務専門官は全国転勤あり → 東京定住を選ぶのは合理的判断。時系列も矛盾なし |
| 注意 | 最初から準備した感を出さない、「ちょっと言葉にしづらいですが…」と前置きで自然に |
| 言葉のニュアンス | 公務員批判にしない（「転勤族の生活は…」より「東京で一緒に暮らせる環境を選びたかった」） |
