# ライフプラン: YouTube 半専業 + バイト戦略

**作成日**: 2026-04-25
**目的**: YouTube チャネルを軸とした半専業ライフプランの財務設計と実行ロードマップ
**前提**: チャネル分析 V2（`analysis/report/INTEGRATED_REPORT_V2.md`）に基づく

---

## ◆ コンセプト

> **「攻めと守りの分離」**
>
> - **守り**: バイト収入で最低限の生活費・社会保険料・税金を賄う
> - **攻め**: YouTube 収入は全額を貯金・投資に回す
>
> この分離により、**心理的圧力なく創作活動が続けられ**、収入の不確実性を投資原資の機会と捉えられる。

---

## 1. 4 つのフェーズ

### Phase 0: 準備期（現在 〜 6-12 ヶ月後）

正社員継続中に行う準備:

- 防衛資金 **12 ヶ月分**（手取り 20 万 × 12 = **240 万円**）の確保
- バイト先候補のリサーチ・面接（在職中）
- 仕込み動画 **4-6 本**を作成
- 開業届・青色申告承認申請書の提出
- 証券口座開設（NISA + iDeCo）
- 会計ソフトの契約・運用練習

### Phase 1: 半専業期（退職直後 〜 5-7 年）

メインの稼働期間:

- バイト 週 28-30 時間（社保適用 / 月 15-17 万円）
- YouTube 月 4-5 本制作（V2 戦略フル稼働）
- フレンとこ② 等の最優先案件を完了
- 劇場版テンプレを 2-3 シリーズ完成
- 投資ポートフォリオ構築（NISA・iDeCo・小規模企業共済）

### Phase 2: 移行期（5-7 年後）

YouTube ネタ切れ感が出始める or 投資ポートフォリオが育った時:

- バイトを 1 本に絞る or 削減
- YouTube は harvest mode 移行準備
- 既存動画の最適化（広告配置 revenue_max、終了画面更新）
- 投資ポートフォリオの取り崩しシミュレーション

### Phase 3: Harvest Mode（7-10 年後 〜）

最終形態:

- YouTube 制作停止 or 月 1 本程度の極小ペース
- 既存動画から年 150-250 万円の受動収入
- 投資ポートフォリオ運用益 + 一部取り崩し
- 余暇・健康・別の創作活動などへ

## 2. 財務構造

### 月次キャッシュフロー（Phase 1 想定）

```
────────────────────────────────────────
収入
  バイト          15-17 万円
  YouTube         20-30 万円（変動）
  ──────────────────────────────
  月収合計        35-47 万円
────────────────────────────────────────
支出
  生活費          12-17 万円  ← バイトで賄う
  公租公課         5-7 万円   ← バイトで賄う
  ──────────────────────────────
  最低必要        17-24 万円
────────────────────────────────────────
余剰（=投資原資）
  毎月の投資額    18-23 万円  ← YouTube 収入分
  年換算          216-276 万円
────────────────────────────────────────
```

### 生活費の内訳目安

| 項目 | 月額 |
|---|---|
| 家賃 | 60,000-80,000 円 |
| 食費 | 30,000-40,000 円 |
| 光熱・通信費 | 15,000-25,000 円 |
| 雑費・娯楽 | 20,000-30,000 円 |
| **生活費合計** | **125,000-175,000 円** |
| 健康保険料 | 15,000-25,000 円 |
| 国民年金 / 厚生年金 | 17,000 円 / 給与控除 |
| 住民税・所得税 | 15,000-25,000 円 |
| **公租公課合計** | **47,000-67,000 円** |
| **総計** | **172,000-242,000 円** |

### バイト要件

| 時給帯 | 必要時間（月17万） | 必要時間（月20万） | 週換算 |
|---|---|---|---|
| 1,100 円 | 154 時間 | 182 時間 | 36-42 時間 |
| **1,300 円** | **131 時間** | **154 時間** | **31-36 時間** |
| 1,500 円 | 113 時間 | 133 時間 | 26-31 時間 |

**推奨選定基準**:
- 時給 **1,300 円以上**
- 週 **30 時間程度**で社保適用
- 動画編集と違う脳の使い方ができる業種
  - 倉庫・物流、図書館スタッフ、カフェ・飲食ホール、品出しレジなど
- シフト柔軟性（YouTube 制作スケジュールに合わせられる）

## 4. 投資ポートフォリオ設計

### 推奨アロケーション（Phase 1）

| 区分 | 比率 | 商品例 |
|---|---|---|
| **NISA つみたて枠** | 27% | eMAXIS Slim 全世界株式 / S&P500 |
| **NISA 成長枠** | 36% | 同上 + 一部高配当 ETF / 成長株 |
| **iDeCo** | 18% | バランス型または株式 100% |
| **小規模企業共済** | 14% | 預金型 |
| **予備現金** | 5% | 普通預金 / 定期 |

### リスク許容度別

#### 攻め（30 代まで）
- 株式比率 **90%**（全世界株式 / S&P500 中心）
- 為替分散（円 / ドル / 全世界）
- 個別株は趣味の範囲で 5-10%

#### バランス（30 代後半 〜 40 代）
- 株式 70% + 債券 20% + REIT 10%
- 配当株 ETF（VYM / HDV）の比率を上げる

### 銘柄推奨（インデックス中心）

| 商品 | 信託報酬 | 性質 |
|---|---|---|
| eMAXIS Slim 全世界株式（オール・カントリー） | 0.058% | 主軸 |
| eMAXIS Slim 米国株式（S&P500） | 0.0814% | サテライト |
| 楽天・全米株式インデックス | 0.162% | 米国一極集中なら |
| iFreeNEXT FANG+ | 0.7755% | 攻めたい場合（配分小） |

| 状況 | 水準 |

| 用途 | 額 | 場所 |
