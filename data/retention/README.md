# 視聴維持率データ取得リクエスト

## 取得方法

YouTube Studio → 該当動画 → **アナリティクス** → **エンゲージメント** タブ
→ **視聴者維持率**グラフ → **︙メニュー** → **データのエクスポート (CSV)**

ダウンロードした CSV を **`<video_id>.csv`** にリネームして
`data/retention/` 直下に保存してください。

スクショの場合は `data/retention/<video_id>.png` で OK（CSV ほどではないが
読み取れます）。

## CSV のリネーム例

ダウンロード時のファイル名（YouTube が付ける名前）はバラバラなので、
**video_id を抜き出してリネーム**してください:

```
audience_retention_xxx.csv → bv2iwq17LVY.csv
table_data.csv             → 8zzMPgZ6bC8.csv
```

参考: `data/retention/REQUEST_LIST.csv` に同じリストが CSV 形式で入っています。
