## 「中の人」の運用方針

> 当初想定の「妻名義で運営 = 中の人=妻のミスリード」は採用しない。**キャラの中の人は明かさない（VTuber 業界標準）**運用に変更。

| 項目 | 方針 |
|---|---|
| キャラの中の人 | **「猫間しおり」というフィクションキャラ**として独立、運営者・声優は明かさない |
| 視聴者からの問い合わせ | 「中の人」「運営者」に関する質問はすべて**ノーコメント** |
| 顔出し | 私・妻ともにしない（既存切り抜きと同様）|
| 声 | 妻のボイス、ただし「キャラの声 = 妻の声」と公にしない |

## 既存切り抜きチャンネルとの分離

> どちらも私名義の事業だが、**ブランド・キャラ・運用は完全分離**。

| 項目 | 既存切り抜き | 新チャンネル「猫間しおり」 |
|---|---|---|
| 名義 | 私 | 私（同じ）|
| Google アカウント | 既存 | **新規 Gmail**（チャンネル分離のため）|
| 銀行口座 | 私の個人口座 | **同じ**（私の事業口座、必要に応じ屋号追加）|
| SNS | 既存 X 等 | **新規 X / note アカウント**、相互フォロー禁止 |
| 端末・ブラウザ | 既存 PC | 同一端末可、**ブラウザのプロファイル分離** |
| 確定申告 | 私の事業所得（合算）| 同じ事業内で処理 |
| キャラ | にじさんじ切り抜き（顔出しなし）| ミニキャラ・妻ボイス |
| ジャンル | VTuber 切り抜き | 投資・節約・ガジェット解説 |

→ 私の本業（AGC 採用想定）の**副業 OK 規定**で問題なし。両チャンネルとも私の副業として運営。
