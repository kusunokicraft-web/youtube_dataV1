## 既存チャンネルとの分離原則

| 項目 | 既存切り抜き | 新チャンネル |
|---|---|---|
| 名義 | 私 | **妻** |
| Google アカウント | 既存 | 新規（妻名義 Gmail）|
| 銀行口座 | 私の個人口座 | **妻名義の新規口座**（or 屋号付き）|
| SNS | 既存 X 等 | 新規アカウント、相互フォロー禁止 |
| 端末・ブラウザ | 既存 PC | 同一端末可、**ブラウザのプロファイル分離** |
| 確定申告 | 私の事業所得 | **妻の事業所得 or 雑所得** |
| キャラ・声 | 顔出しなし、声 = 私 | ミニキャラ、声 = 妻 |

→ 「私と妻は別人格として運営、夫は無償の編集ボランティア」が外形的事実になるよう設計。
