## 1. キャラクター名（暫定確定）

| 項目 | 内容 |
|---|---|
| **チャンネル / キャラ名** | **猫間 しおり** |
| 読み | ねこま しおり |
| ローマ字表記 | Nekoma Shiori |
| 拍数 | 6 拍（呼びやすい）|
| ニックネーム候補 | しおりちゃん / しおりん / しおりさん / ねこまさん |
| 確定理由 | 猫モチーフ + 栞（読書）+ おっとり感 + 親しみ + 5-6 拍 + 被りなし |

### 命名の由来（公開用ストーリー）

> 「**猫の間（あいだ）に栞を挟むように、暮らしのちょっとした隙間に投資・節約・ガジェットの知恵を挟み込む**」というコンセプト。
> 苗字「猫間」は大阪の歴史地名「猫間川」由来。名前「しおり」は本のしおり = 読書好きで知識を集めるイメージ。

## 3. 性格・人格（詳細版）

### 3-1. 基本トーン

| 軸 | 設定 |
|---|---|
| **基本トーン** | **おっとり・癒し・まじめ・親しみ・かわいい**（5 要素のバランス）|
| 競合分析からの方向性 | **千両路りあ系（落ち着き）と豹矢りいす系（凡人感）の中間**、おっとり OL |
| 第一印象 | のんびりしてるけど話すと意外としっかりしてる |
| ギャップ | **おっとり見えて数字の話になると急に的確**（投資・節約解説の説得力）|

### 3-2. 性格の構成要素

| 要素 | 比率 | 内容 |
|---|---|---|
| **おっとり** | 30% | のんびりした口調・反応がやや遅め・落ち着いた声 |
| **真面目** | 25% | 数字・データに正確、調べ物が好き、独学を続ける |
| **親しみやすい** | 20% | 「みなさん」呼び・丁寧・穏やか |
| **かわいい** | 15% | 「ふふ」笑い・小さな失敗・素直なリアクション |
| **意外性（ギャップ）**| 10% | 投資・節約の数字を語る時の的確さ・冷静さ |

### 3-3. 視聴者への接し方

| 場面 | 接し方 |
|---|---|
| 視聴者呼称 | **「みなさん」**（「〇〇さん」呼びでなく統一）|
| 距離感 | 「教える先生」ではなく「**一緒に学ぶ友達**」 |
| 質問対応 | 「いい質問ですね」「うんうん、わかります」と肯定から入る |
| 失敗の共有 | 「実はわたしも…」と自分の失敗談を交える |
| 専門用語 | 必ず「これは〇〇という意味なんですよ」と噛み砕く |

### 3-4. 設定の最小限ライン（再確認）

| 項目 | 設定 |
|---|---|
| キャラの本質 | **独学で投資・節約・ガジェット・ライフハックを学んだ普通の OL**（ぽんちよ式設定の VTuber 化）|
| なぜ詳しいか | 大学時代に家計簿アプリで貯金が増えて以来、お金とガジェットの研究が趣味に |
| 設定の濃さ | 軽め、「世界観」より「人柄」で勝負 |

## 4. 口調・話し方（詳細版）

### 4-1. 基本フレーズ

| 場面 | フレーズ例 |
|---|---|
| **一人称** | 基本「**わたし**」、稀に「**しおり**」（自己愛感が薄め）|
| **二人称** | 「**みなさん**」「あなた」 |
| **挨拶** | 「みなさん、こんにちは。猫間しおりです」 |
| **語尾** | 「〜なんですよ」「〜ですね」「〜してみてくださいね」 |
| **相槌** | 「ふふ」「ふむふむ」「うんうん」「なるほど〜」 |
| **強調** | 「これがけっこう…**大事なんですよ**」（一拍置く） |
| **驚き** | 「えっ！」「まあ！」「あら？」 |
| **困惑** | 「えっと…」「むずかしいですね」 |
| **失敗** | 「あちゃ〜」「失敗しちゃいました」「ふふ、やっちゃいました」 |
| **喜び** | 「やった〜」「ふふっ、できました」 |
| **締め** | 「それでは、また」「お役に立てたらうれしいです」 |

### 4-2. 福岡要素（隠し味）

| 観点 | 方針 |
|---|---|
| 出身設定 | 福岡県出身（character_design 既存設定）|
| 博多弁の使用頻度 | **基本は標準語、ごくたまに「〜と」「〜やね」程度の隠し味** |
| 例 | 「これ、安くていいんですよ。**ね**」 程度に留める |
| 出さなくても OK | 普段は標準語のおっとり OL、博多弁は親密シーンで偶発的に |

### 4-3. NG な口調・表現

| NG | 理由 |
|---|---|
| 「お前」「あなた様」| キャラのトーンに合わない |
| 「〜だぜ」「〜さ」| 男性的すぎる |
| 「ですわ」「ですことよ」| お嬢様キャラに偏る |
| 「ヤバい」「マジで」| カジュアルすぎる、おっとり感喪失 |
| 「絶対」「必ず」| 投資解説で断定は避ける（信頼性宣言と矛盾）|
| 「素人」「初心者向けに」| 視聴者を見下す表現 |

## 4-4. 場面別 台詞サンプル集（10 例）

実際の動画で使う想定の台詞。妻が読む際の参考。

### サンプル 1: オープニング（毎回共通の挨拶）

> 「みなさん、こんにちは。猫間しおりです。今日もお金とガジェットの話、ふふ、お届けしますね」

### サンプル 2: 投資解説（NISA）

> 「NISA、ですか。これがけっこう…**大事なんですよ**。年間 360 万円まで非課税で投資できる、すごい制度なんです。わたしも毎月コツコツ、続けてます」

### サンプル 3: 節約術（固定費見直し）

> 「固定費、見直してみましたか？ 携帯料金、保険、サブスク…ふむふむ。実はわたし、これで月 1 万 5 千円、浮いたんですよ。やってみるとびっくりです」

### サンプル 4: ガジェットレビュー（時短家電）

> 「ロボット掃除機、買ってみました。最初は『高いなぁ…』って迷ったんですけど、ふふ、もう手放せません。掃除の時間、本当に減るんです」

### サンプル 5: もしも話シミュレーション

> 「もし、25 歳から月 5 万円、NISA で積み立てたら…ふむふむ、計算してみますね。10 年で約 770 万円。これは、けっこう…大きいですよ」

### サンプル 6: 失敗談（自分の経験）

> 「実はわたし、初めての投資で大失敗しちゃいまして…ふふ、損切りできなくて。みなさん、損切りラインは決めておいてくださいね。あちゃ〜」

### サンプル 7: 視聴者への質問（コメント誘導）

> 「みなさんは、どんな節約術、されてますか？ よかったら、コメント欄で教えてください。わたしも勉強したいので、ふふ」

### サンプル 8: 信頼性宣言（初回・概要欄）

> 「このチャンネルでは、投資勧誘や情報商材のご紹介は一切しません。LINE 登録もありません。安心して、見ていってくださいね」

### サンプル 9: 困った時のリアクション

> 「えっと…むずかしいですね。これ、わたしも最初わからなかったんです。一緒に、ゆっくり見ていきましょう」

### サンプル 10: エンディング（毎回共通の締め）

> 「今日はここまで。**みなさんの暮らしの隙間に、ちいさな知恵が挟まりますように**。それでは、また」

→ オープニングとエンディングは**毎回固定の決め台詞**として、視聴者の習慣化を狙う（両学長「お金にまつわる 5 つの力」式）。

## 5. ビジュアル方針（発注前の指針）

| 項目 | 方針 |
|---|---|
| 頭身 | **2.5-3 頭身のミニキャラ** |
| 性的魅力 | **抑える**（一般受け重視）|
| 髪 | ボブ〜ミディアム、地味色（茶 / 黒 / 灰）|
| 目 | 大きめ・優しい、淡いブラウン or グリーン |
| 服装 | カジュアル OL（ニットカーディガン + シンプルスカート）|
| 猫要素 | **猫耳 + しっぽ**（控えめ）、肉球モチーフのアクセサリー |
| 全体雰囲気 | やわらかい線・暖色寄り（豹矢りいす系の優しさ）|
| **避けるべき方向** | ニャンコ先生風（白茶ぶち丸身）/ 派手色 / 露出多め |
