# 新チャンネル「猫間しおり」競合分析

> 作成日: 2026-04-30
> 対象チャンネル: 猫間しおり（仮称・暫定確定）
> 想定コンセプト: 動画型 × ミニキャラ × 投資・節約・ガジェット・ライフハック × 妻ボイス
> 関連: `business_plan.md §4「差別化戦略」` / `character_design.md` / `simulations.md`

## 1. 概要・調査目的

`business_plan.md §4` で粗く整理した競合マップ（顔出し金融 / ゆっくり解説 / FP 系 VTuber / 解説系 VTuber / 雑談系 VTuber）を、**実在チャンネル名 + 登録者数 + スタイル分析 + 差別化軸 + パクれる要素**まで踏み込んで定量化する。

### 1-1. 調査の目的

| # | 目的 | 出力 |
|---|---|---|
| 1 | **空白ニッチの可視化** | 「動画型 × ミニキャラ × 投資+ガジェット融合」の空白を定量証明 |
| 2 | **差別化軸の確定** | 「猫間しおり」が他チャネルと識別できる 3-5 個の独自要素 |
| 3 | **パクれる要素の抽出** | 各競合から学ぶ運用ノウハウ（信頼性宣言・サムネ方針・口調・シリーズ化等）|
| 4 | **KPI 目安の妥当性検証** | Year 1 末 500 / Year 5 25k 登録という `simulations.md` 中立シナリオが現実的か |
| 5 | **ポジショニングマップ** | 視覚的にチャネルの位置を確定 |

### 1-2. 調査の前提

- 直接競合（同一フォーマット）は実は薄いとの仮説 → 検証
- ジャンル特化型（千両路りあ系）/ スタイル特化型（豹矢りいす系）/ 顔出し系（両学長系）を分けて分析
- 既出の WebSearch 結果（過去会話で蓄積）を活用、不足分は追加検索

## 2. 競合分類フレームワーク

### 2-1. 3 層構造

```
[直接競合]   動画型 × 解説 × 金融・節約系（コンテンツ形式が同一）
   │
[近接競合]   VTuber 業界全般（キャラ駆動という共通点）
   ├─ 配信中心型 VTuber
   └─ ガジェット系 VTuber
   │
[参考]       顔出し金融 YouTuber（キャラ駆動 + 解説型として参考）
```

### 2-2. 各層の定義

| 層 | 定義 | 競合度 |
|---|---|---|
| **直接競合** | 同じ動画フォーマット（スライド + ナレ + キャラ）+ 同じジャンル（金融・節約）| ◎ 高 |
| **近接 VTuber** | キャラ駆動だがフォーマット or ジャンルが異なる | ○ 中 |
| **参考** | フォーマット類似（解説型動画）だが顔出しでキャラ駆動ではない | △ 低 |

### 2-3. 評価軸（各チャネルを以下の軸で記述）

| 軸 | 内容 |
|---|---|
| 登録者数 | 直近の YouTube チャネル登録者 |
| 動画スタイル | 形式（スライド / 配信切り抜き / ホワイトボード等）|
| 投稿頻度 | 月本数・配信頻度 |
| キャラ設定 | 視覚的キャラの濃さ・声（人間 / ボイロ / ゆっくり）|
| ジャンル特化度 | 単一テーマ vs 横断 |
| 編集品質 | サムネ・テロップ・字幕の作り込み |
| 強み | 訴求軸・成功要因 |
| 弱み | 構造的な天井・差別化が薄い点 |

### 2-4. 「猫間しおり」の暫定ポジション（仮説）

| 要素 | 設定 |
|---|---|
| 動画型 | ◎（配信ではない・検索 SEO 重視）|
| キャラ駆動度 | 中（半分情報・半分キャラ）|
| ジャンル | 投資 50% + 節約 + ガジェット + ライフハック の融合 |
| 声 | **人間ボイス（妻）**、ボイロ・ゆっくりではない |
| キャラデザ | ミニキャラ・性的魅力少・親しみ重視 |
| 訴求層 | 25-35 歳・男性 65% / 女性 35% / 年収 350-700 万 |

→ 詳細は §6-7 のポジショニングマップで定量化。
