# 別 AI ブレスト続行プロンプト集

> ChatGPT / Gemini / Claude 別セッション等にコピペで投げて、追加候補をブレストする用のプロンプト集。

## 1. 最終版・季語+古語幼名前（最も完成度高い）

```
日本の VTuber 命名コンサルタントとして、以下の制約で候補を 30 案ブレストしてください。

# 必須条件
1. 構造: 3 拍ひらがな苗字 + 2 拍ひらがな名前 = 5 拍ぴったり
2. ひらがな統一
3. 苗字は 3 拍ひらがな季語（春夏秋冬+通季・古語の自然詩語）
4. 名前は 2 拍ひらがなの古語的かつ幼さもある語
5. 「うい」「しぐれ」「ふみ」「ちあき」「ちとせ」禁止
6. 苗字は現実に稀少（エゴサ回避）
7. 苗字に「数字・期間・お金・神話・骨董・地名特化・学問」連想禁止

# 苗字の例（参考）
うらら / おぼろ / かすみ / すみれ / やよい / すずな / なずな / つばめ / ひばり /
ほたる / なぎさ / みなも / もみじ / このは / すすき / つきみ / かんな / きつね /
ゆきみ / ふぶき / しもや / しずく / みおり / かのこ / しのぶ

# 名前の例（参考）
いと / まな / あや / たえ / もえ / まゆ / きく / すず / かや /
のの / やよ / ねね / ちさ / おと / たま / まり / のえ / なお / あい / ゆい / りん

# キャラ性
- 26 歳 OL・等身大・おっとり・親しみ
- 動画型・複数ジャンル横断（投資・節約・ガジェット・時事・学術系）
- N1 ペルソナ「VTuber 好き男性社会人・知識欲・心地よいトーン」適合

# 既出強候補（被り避ける）
- ★ 最終候補圏: かすみあや / すずなおと
- □ 参考保持: ちあきゆい / ちとせゆい / いずもさや / いまりさや
- 推奨: おぼろいと / おぼろまな / このはおと / ほたるもえ /
  うららのえ / ほたるまな / ゆきみまな / かすみまな

# 期待
- 季語ベースの 30 案以上、各候補に「季節 / しぐれうい度 / 響きの良さ」採点
- トップ 10 を選定
```
